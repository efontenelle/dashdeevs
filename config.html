Configuração

Informe suas credenciais do Azure DevOps. Org e projeto ficam em localStorage; o PAT fica em sessionStorage (apagado ao fechar a aba). Nada é enviado para outro servidor.

Exemplo: para https://dev.azure.com/minha-empresa, use minha-empresa.
Escopos mínimos: Code: Read, Work Items: Read, Project: Read.

Como gerar o PAT

  1. Abra dev.azure.com e faça login.
  2. Clique no avatar → Security → Personal access tokens.
  3. Clique em New Token, defina um nome e expiração.
  4. Selecione os escopos: Code (Read), Work Items (Read), Project and Team (Read).
  5. Copie o token e cole aqui.
